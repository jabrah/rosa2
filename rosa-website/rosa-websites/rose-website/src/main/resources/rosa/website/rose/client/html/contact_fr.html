Réactions à propos du site Internet

Vous pouvez contribuer à l'amélioration de ce site Internet. Vos corrections, signalement d'erreurs de traduction ou transcriptions de feuillets additionnels sont les bienvenus. Veuillez nous contacter à l'adresse électronique suivante : contactus@romandelarose.org pour tout problème concernant le contenu du site ou pour l'ajout de nouveaux supports.

Contacter le webmestre

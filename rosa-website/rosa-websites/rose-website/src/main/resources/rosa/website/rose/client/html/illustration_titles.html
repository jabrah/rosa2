Below is a list of titles for illustrations in Rose manuscripts. These titles are in English, but the character names in the titles are mostly in French (a list of character names and variants has also been provided). Miniatures can be found by searching for the illustration title in the advanced search tool.

The "position" column shows the average page number of illustrations with that title. The "frequency" column shows the number of times an illustration with that title occurs in all the manuscripts in our collection.
